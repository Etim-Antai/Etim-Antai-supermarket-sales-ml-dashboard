🌲 Random Forest Regression — Model Summary

This Machine Learning model predicts supermarket sales using historical transaction records. The dataset was preprocessed using an OrdinalEncoder inside a ColumnTransformer, while numerical variables were passed through unchanged. Four regression algorithms were trained and evaluated using an 80/20 train-test split.
